# Contributing to BullBearPK

Thank you for your interest in contributing to BullBearPK! We welcome contributions from the community to help make this AI-driven investment platform even better.

## How to Contribute

### 1. Reporting Bugs
- Use the GitHub Issues tab to report bugs.
- Provide a clear description of the issue and steps to reproduce.

### 2. Feature Requests
- Open an issue with the [Feature] tag to propose new agents or UI enhancements.

### 3. Pull Requests
- Fork the repository.
- Create a new branch for your feature (`git checkout -b feature/amazing-feature`).
- Commit your changes (`git commit -m 'Add amazing feature'`).
- Push to the branch (`git push origin feature/amazing-feature`).
- Open a Pull Request.

## Development Setup
- Follow the instructions in `README.md` and use `setup_project.py` to prepare your environment.
- Ensure your code follows PEP 8 guidelines for Python and uses TypeScript for Frontend.
